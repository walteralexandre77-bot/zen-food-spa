# 🍣 Zen Food SPA

Projeto de uma página institucional **SPA (Single Page Application)** responsiva desenvolvido com **HTML5, CSS3 e JavaScript**, simulando o site de um restaurante japonês fictício chamado **Zen Food**.

O objetivo do projeto é demonstrar a construção de uma landing page moderna e reutilizável para micro, pequenas e médias empresas.

---

## 📌 Sobre o Projeto

Este projeto apresenta uma interface web de página única com navegação interna suave entre seções e layout responsivo adaptado para dispositivos móveis, tablets e desktops.

A aplicação foi estruturada utilizando boas práticas de organização de código front-end e semântica HTML.

O site contém:

- Header fixo com menu de navegação
- Seção Home (Hero)
- Seção Sobre
- Seção Serviços
- Seção Depoimentos
- Seção Contato com formulário
- Footer institucional
